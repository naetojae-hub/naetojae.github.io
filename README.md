# Naetoja Ebere Ezerioha — Portfolio

## Files
- index.html — page content (edit projects, contact links, bio text here)
- style.css — visual design
- script.js — terminal typing animation on the hero
- assets/profile.png — your profile photo

## To customize
1. Open index.html in any text editor.
2. Replace the 3 placeholder project cards (title, description, tool tag, link) with your real class projects.
3. Replace the skill bar percentages under `<section id="skills">` if you want them to reflect your actual comfort level.
4. Replace the contact links (email, LinkedIn, GitHub) under `<section id="contact">`.

## To preview locally
Just double-click index.html — it opens directly in your browser, no server needed.

## To publish for free (GitHub Pages)
1. Create a free GitHub account at github.com if you don't have one.
2. Create a new repository (e.g. "portfolio") — keep it Public.
3. Upload index.html, style.css, script.js, and the assets folder to that repo (use "Add file" > "Upload files" on the repo page — no command line needed).
4. In the repo, go to Settings > Pages.
5. Under "Build and deployment", set Source to "Deploy from a branch", branch = main, folder = / (root). Save.
6. Wait 1–2 minutes, then refresh that Pages settings page — it will show your live link, something like:
   https://your-username.github.io/portfolio/
